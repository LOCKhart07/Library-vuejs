## Run Locally

Clone the project

```bash
  git clone https://github.com/LOCKhart07/Library-vuejs.git
```

Go to the project directory

```bash
  cd library-vuejs
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
